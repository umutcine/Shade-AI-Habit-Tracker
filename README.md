**Shade** is not your typical friendly habit tracker. It utilizes **OpenAI (GPT-4)** to create a toxic yet humorous persona that "roasts" you when you fail your habits. Built fully natively with **SwiftUI** and backed by **Firebase**.

---

### üõ† Tech Stack

* **Language:** Swift 5
* **UI Framework:** SwiftUI
* **Architecture:** MVVM
* **Backend & Database:** Firebase (Firestore, Auth, Cloud Functions)
* **AI Integration:** OpenAI API (GPT-4 Turbo) with custom Prompt Engineering
* **Monetization:** RevenueCat (Subscriptions & Paywalls)
* **Dependency Manager:** Swift Package Manager (SPM)

### üí° Key Features Implemented

* **Dynamic AI Persona:** Developed a custom AI logic that analyzes user data to generate context-aware, humorous feedback.
* **Real-time Sync:** Utilized **Firebase Firestore** for seamless data synchronization across devices.
* **Smart Notifications:** Implemented local notification triggers based on user inactivity.
* **Paywall System:** Integrated remote-config paywalls managed via RevenueCat.
